Defining an HTML Table

An HTML table is defined with the tag.

Each table row is defined with the

tag. A table header is defined with the <.th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <.td> tag.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

HTML Table - Adding a Border

If you do not specify a border for the table, it will be displayed without borders.

A border is set using the CSS border property:

table, th, td { border: 1px solid black; }
